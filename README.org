* Overview

salm.lisp is a Serverless Application Lisp Model - a framework for building serverless applications using Lisp.

* Installation

#+begin_src shell
# Clone the repository
git clone https://github.com/conao3/salm.lisp.git
cd salm.lisp
#+end_src

* Usage

#+begin_src shell
make build
#+end_src
